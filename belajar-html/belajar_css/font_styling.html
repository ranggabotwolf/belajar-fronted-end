Point Penting menggunakan css font styling

Perbedaan penggunaan antara font styling dan text styling adalah :

  • Jika menggunakan font styling pada css, maka akan mengatur semua yang berkaitan dengan styling pada font
  • Sementara menggunakan text styling pada css, akan mengatur semua yang berkaitan dengan styling pada text

Perlu di ingat!!!

Font family serif adalah font yang mempunyai kaki / lengkukan tambahan pada tiap ujung huruf, sementara sans-serif adalah font yang tidak mempunyai kaki / lengkukan tambahan pada tiap ujung huruf
