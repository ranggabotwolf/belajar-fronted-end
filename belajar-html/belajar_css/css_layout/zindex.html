Kotak yang berwarna hijau diatas berada di paling atas posisi nya, karena dideklarasikan paling terakhir

Ada cara agar kita bisa mengatur elemen apa yang ingin kita tampilkan di paling depan layar tanpa harus mengubah urutan deklarasi dari elemen tersebut, yaitu menggunakan z-index.

Semakin besar nilai z-index, maka akan semakin dekat dengan user / tampil paling depan, begitu juga sebaliknya, semakin kecil nilai z-index suatu elemen, maka akan berada semakin jauh dari tampilan ke user
