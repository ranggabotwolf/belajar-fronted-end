Catatan Penting!

  • Padding tidak bisa diberikan nilai negatif atau auto, kenapa? karena padding adalah batas transparan antara content dan border, dan dalam setiap box model, pasti sebuah konten pasti dibungkus oleh padding, maka dari itu padding tidak bisa diberikan nilai negatif, karena content tidak akan bisa melewati batas padding, karena padding merupakan pembungkus content, dan akan bernilai 0 by default yang mengakibatkan content pasti selalu ditengah by default, makanya tidak bisa diberikan nilai auto pada padding
  • Ukuran padding berpengaruh pada ukuran box, jadi ukuran padding akan menambah ukuran box dari sebuah box model, jadi hati-hati, harap perhitungkan matang-matang masing-masing nilai dari content, padding, maupun border

  • Ukuran border juga berpengaruh pada ukuran box, jadi ukuran box akan bertambah juga sejumlah tebal border
  • Jadi, rumus dimensi box adalah, Total Dimensi = width/height + Padding + Border
