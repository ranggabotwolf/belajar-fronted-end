Terlihat dari box div diatas, tidak terlihat ada perubahan walaupun sudah diberi top 30px, karena memang position default sebuah element adalah static. Lihatlah perbandingan dengan box div dibawah yang menggunakan position relative

Sementara box div diatas terlihat jelas bahwa box berwarna hijau telah bergerak sesuai yang diinputkan oleh si pemrogram

Position relative bisa kita gerakkan posisinya menggunakan properti top, left, right & bottom (ruang yang ditempati oleh elemen tersebut masih ada), dan akan bergerak relatif terhadap posisi semulanya, jadi jika kita set top: 0px dan left: 0px maka elemen tidak akan berubah posisi
