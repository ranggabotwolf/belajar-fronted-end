bila file berada dalam folder yang sama dengan file hyperlink, maka gunakan href="nama_folder/nama_file"
tapi file hyperlink berada diatas dari file yang sedang dibuka, maka gunakan href="../nama_file"
.. (titik 2) pada href artinya untuk naik 1 folder, lalu mencari nama file yang kita ingin gunakan / tuju
