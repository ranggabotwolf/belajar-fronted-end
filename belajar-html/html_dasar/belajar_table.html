Tabel Simpel

baris 1, kolom 1 baris 1, kolom 2
baris 2, kolom 1 baris 2, kolom 2

Perlu diperhatikan

  • border : Memberikan batas / garis di luar cell (menjadi garis tabel)
  • cellspacing : Memberikan jarak antar cell
  • cellpadding : Memberikan luas / isi pada cell
Ketiga atribut diatas lebih baik digunakan dengan css

Perlu diperhatikan

  • Jika ingin menggunakan Colspan,kita harus menghapus td berikutnya sesuai jumlah cell kolom yang mau kita gabungkan dengan colspan
  • Jika ingin menggunakan Rowspan, kita harus menghapus td pada tiap" tr sesuai jumlah cell baris yang mau kita gabungkan dengan rowspan
